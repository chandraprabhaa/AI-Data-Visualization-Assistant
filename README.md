# 📊 AI Data Visualization Assistant

AI-powered data analysis application that allows users to upload CSV datasets and generate intelligent insights using Groq API and Llama 3.3 70B model.

Built with Streamlit and Pandas to provide dataset preview, basic dataset information, and AI-based analysis through an interactive web interface.

---

## ✨ Features

* 👀 Dataset Preview
* 📋 Basic Dataset Information
* 🤖 AI Generated Insights

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Pandas
* Groq API
